# Hero's Adventure
Một tựa game hành động – nhập vai thế giới mở được tôi phát triển bằng Unity, tập trung vào chiến đấu, khám phá và nâng cấp nhân vật.

## Screenshots
![Gameplay](https://lucastapersonal-fzwaj.wordpress.com/wp-content/uploads/2025/11/beach.png)

## Features
- Combat action với Animation Event
- Hệ thống Enemy AI (Patrol, Chase, Combat)
- Quest system
- Item, vũ khí, kỹ năng
- World exploration

## Installation
1. Clone repo  
2. Mở bằng Unity Editor (phiên bản đề xuất)  
3. Run scene Main
